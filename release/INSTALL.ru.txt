================================================================================
  BELARUS CHAMP CLICKER — Инструкция по установке (Windows 64-bit)
================================================================================

Спасибо, что скачали Belarus Champ Clicker.

В этом пакете есть всё необходимое. Дополнительно вручную нужно установить
только драйвер ввода Windows — один раз (см. ниже).


БЫСТРАЯ УСТАНОВКА (рекомендуется)
---------------------------------

1. Распакуйте весь ZIP-архив в удобную папку
   (например: Загрузки\BelarusChampClicker).

2. Дважды щёлкните по файлу  Install.cmd
   (или щёлкните правой кнопкой по Install.ps1 → «Выполнить с помощью PowerShell»).

3. Если Windows запросит права администратора — нажмите «Да».
   Так устанавливается драйвер ввода, нужный для виртуальных кликов.

4. Если установщик сообщит о необходимости перезагрузки — перезагрузите компьютер.
   Это нужно только при первой установке на новом ПК.

5. После перезагрузки откройте  Belarus Champ Clicker.exe  через:
     - ярлык на рабочем столе (создаётся установщиком), или
     - меню «Пуск» → «Belarus Champ Clicker», или
     - эту папку (если запускаете без установки).

6. В программе:
     a) Введите код активации при первом запуске (см. АКТИВАЦИЯ)
     b) Нажмите  Start  (Старт)
     c) Нажмите  Add key...  и нажмите клавишу-триггер
     d) При необходимости задайте  Delay (ms)
     e) Запустите игру и удерживайте клавишу-триггер


АКТИВАЦИЯ
---------

  При первом запуске программа запросит код активации.

  1. Скопируйте «Computer ID» из окна активации
  2. Отправьте его продавцу, чтобы получить код
  3. Вставьте код и нажмите Activate (Активировать)

  Один код — один компьютер.


ПОРТАТИВНЫЙ ЗАПУСК (без установщика)
------------------------------------

Можно запускать  Belarus Champ Clicker.exe  прямо из этой папки, не запуская
Install.cmd — но драйвер ввода всё равно нужно установить один раз (через Install.cmd).


КАК ПОЛЬЗОВАТЬСЯ
----------------

  Start     — включить кликер (делайте это до запуска игры)
  Stop      — выключить кликер

  Удерживайте клавишу-триггер — повторяется цикл нажатий.
  Отпустите триггер — текущий цикл завершится, цикл остановится.


РЕШЕНИЕ ПРОБЛЕМ
---------------

  При нажатии Start появляется «Setup required»
      Запустите Install.cmd от имени администратора, затем перезагрузите ПК.

  Клики не срабатывают в игре
      Запускайте кликер до запуска игры.
      Попробуйте задержку 50 мс или больше.

  Программа не запускается
      SmartScreen может предупредить при первом запуске — «Подробнее» → «Выполнить в любом случае»,
      если вы скачали архив из надёжного источника.

  Удаление
      Дважды щёлкните Uninstall.cmd в этой папке.
      (Удалит программу, ярлыки и активацию. Драйвер USBip — отдельно в Параметрах Windows.)
