  BELARUS CHAMP CLICKER — Лицензии (только для автора, не отдавать пользователям)
================================================================================

Один код — один компьютер. Консоль не нужна.


ОДНОРАЗОВАЯ НАСТРОЙКА
---------------------

1. Запустите:

     clicker\build-author.ps1

   Появится  licgen-gui.exe

2. Откройте  licgen-gui.exe  →  Create signing keys
   (программа и ZIP для пользователей соберутся автоматически)

3. Продолжите Шаг 2 в том же окне


ВЫДАТЬ КОД ПОКУПАТЕЛЮ
---------------------

1. Покупатель запускает кликер, нажимает  Copy ID / Копировать  и присылает ID.

2. Откройте  licgen-gui.exe

3. Вставьте ID →  Generate code  →  Copy code

4. Отправьте код покупателю. Он вставляет его в окно активации.


ДЛЯ СВОЕГО ПК
-------------

В licgen-gui.exe:  Use my ID  →  Generate code  → вставьте код в кликер.


ЗАМЕТКИ
-------

- licgen-gui.exe не кладите в ZIP для пользователей.
- license\private.pem храните в секрете.
