  BELARUS CHAMP CLICKER — License tool (author only, do not ship to users)
================================================================================

Each customer gets one activation code tied to their computer.
The same code will NOT work on a second PC.

Use the GUI tool — no console required.


ONE-TIME SETUP
--------------

1. Double-click or run:

     clicker\build-author.ps1

   This creates  licgen-gui.exe

2. Open  licgen-gui.exe  and click  Create signing keys
   (builds the user app and release ZIP automatically)

3. Continue to Step 2 in the same window to issue codes


ISSUE A CODE FOR A CUSTOMER
---------------------------

1. Customer runs Belarus Champ Clicker once.
   They click  Copy ID  in the activation window and send you the ID.

2. Open  licgen-gui.exe

3. Paste their Computer ID, click  Generate code, then  Copy code

4. Send the code to the customer. They paste it in the app and click Activate.


FOR YOUR OWN PC
---------------

In licgen-gui.exe click  Use my ID  →  Generate code  → paste into the clicker app.


NOTES
-----

- Do not include licgen-gui.exe in the user ZIP.
- Keep license\private.pem secret.
